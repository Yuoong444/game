# 마법봉 강화 RPG v63

모바일/PC에서 바로 실행 가능한 HTML/CSS/JS 기반 마법봉 강화 게임입니다.  
v63은 환생 시스템 없이도 반복 플레이가 살아나도록 **크리티컬, 콤보, 성장 목표, 옵션 풀 확장, 핵심 HUD 재배치**를 적용한 중독성 게임 구조 개선 버전입니다.

## 실행 방법

1. ZIP 파일 압축을 풉니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. GitHub Pages에 올릴 때는 압축을 푼 파일 전체를 저장소 루트에 업로드합니다.

## 파일 구조

```text
magic_wand_rpg_v63/

```

## v63 주요 개선사항

### 1. 실제 게임 루프 개선

기존 흐름은 `클릭 → 강화 → 반복`에 가까웠습니다.  
v63은 아래 구조로 개선했습니다.

```text
클릭/자동 수익
→ 크리티컬/콤보 보상
→ 골드/보석/마력석 획득
→ 강화/옵션 파밍
→ 성장 목표 보상
→ 더 빠른 성장
```

### 2. 핵심 HUD 재배치

모바일에서도 바로 확인할 수 있도록 핵심 수치를 메인 화면에 표시했습니다.

- 클릭 수익
- 초당 수익(DPS)
- 크리티컬 확률
- 콤보 배율
- 보석 확정 보상까지 남은 골드
- 강화 성공률

### 3. 크리티컬 시스템 추가

클릭 채굴 시 확률적으로 크리티컬이 발생합니다.

- 등급이 높을수록 기본 크리 확률 증가
- 화염/암흑 속성은 크리 쪽에 추가 이점
- 크리티컬 피해 옵션으로 배율 증가
- 크리티컬 발생 시 별도 메시지와 연출 표시

### 4. 콤보 시스템 추가

짧은 시간 안에 연속 터치하면 콤보 배율이 증가합니다.

- 빠른 클릭에 보상 제공
- 콤보 배율은 메인 HUD에 표시
- 클릭 채굴 손맛 강화

### 5. 성장 목표 시스템 추가

환생 없이도 매일/매 세션 목표가 생기도록 성장 목표 UI를 추가했습니다.

예시 목표:

- 마법봉 30회 채굴
- 크리티컬 5회 터뜨리기
- 강화 3회 성공
- 자동 수익 20,000골드 누적

목표 완료 시 골드, 보석, 마력석 보상을 받을 수 있습니다.

### 6. 랜덤 옵션 구조 확장

옵션 풀을 확장했습니다.

- 클릭 수익 증가
- 자동 수익 증가
- 강화 성공률 보너스
- 골드 획득량 증가
- 보석 획득량 증가
- 크리티컬 확률
- 크리티컬 피해

옵션은 가중치 기반으로 등장하며, 같은 등급의 마법봉이라도 성격이 달라지도록 구성했습니다.

### 7. 데이터 구조 변경

저장 키가 변경되었습니다.

```js
SAVE_KEY = "magicWandRpgSaveV63"
SAVE_VERSION = 630
```

v62 저장 데이터는 자동으로 불러와 v63 구조로 마이그레이션됩니다.

추가 저장 항목:

- `totalClicks`
- `totalCrits`
- `bestCrit`
- `todayQuestIndex`
- `todayQuestProgress`
- `todayQuestClaimed`
- `lastQuestKey`

### 8. UI/UX 개선

- 메인 화면에 성장 목표 카드 추가
- 현재 무기 정보에 초당 수익, 크리티컬, 전투력 표시
- 크리티컬 발생 시 시각 효과 추가
- 모바일에서 핵심 수치가 잘리지 않도록 반응형 보정

- 화염 속성 LV7 기준 실제 배율이 의도(×1.18)의 제곱(×1.39)으로 계산되던 버그 수정
- `calculateIncome()` 에서 중복 호출 제거

**2. 초기화 버튼 v63 상태 누락 수정**
- `resetBtn` 클릭 시 `totalClicks`, `totalCrits`, `bestCrit`, `clickCombo`, `lastClickTime`, 퀘스트 관련 5개 변수가 초기화되지 않던 문제 수정

### 🟡 퀘스트 시스템 개선

**3. 퀘스트 자동 완료 방식으로 변경**
- 기존: 목표 달성 후 버튼을 눌러야 보상 수령
- 변경: 목표 달성 순간 자동으로 보상 지급 + 화면 이펙트 + 메시지 표시
- 수동 보상 버튼 제거 (게임이 직접 알려줌)

**4. 퀘스트 선정 로직 개선**
- 초반(자동수익 0)에는 "자동 수익 20,000골드 누적" 퀘스트 제외
- 어제와 같은 퀘스트가 연속으로 나오지 않도록 중복 방지 적용

### 🟢 UI 개선

**5. compact-stats HUD 크리+콤보 셀 분리**
- 크리티컬 확률과 콤보 배율이 같은 셀에 혼재되어 의미 파악이 어려웠던 문제 해결
- `크리확률` 셀과 `콤보` 셀을 독립 분리 (5열 → 5열 유지, 레이블 명확화)

**6. 퀘스트 카드 밝은 테마 대응**
- 어두운 배경 + 흰 텍스트 고정으로 밝은 테마에서 튀던 문제 수정
- 다크/라이트 테마 각각 적합한 배경/텍스트 색상 적용
- 퀘스트 완료 시 초록 테두리 + 바 색상 변경으로 완료 상태 시각화

## v64 메뉴/팝업 그리드 안정화 패치
- 메뉴 버튼 또는 우측 탭으로 창을 열 때 패널이 우측 메뉴 그리드 안에 끼어 들어가며 깨지는 문제를 방지했습니다.
- PC에서는 우측 탭을 세로 메뉴로 고정하고, 도감/옵션/뽑기/가방 패널은 중앙 팝업 오버레이로 열리도록 정리했습니다.
- 모바일에서는 메뉴 팝업, 패널 팝업, 도감/필터/가방 그리드가 화면 폭을 넘지 않도록 `minmax(0, 1fr)` 기반으로 보정했습니다.
