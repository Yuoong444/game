## 0.69.0 업데이트 (게임 구조 개편)

### 핵심 변경사항
- 버전 표기 방식을 `0.x.y`로 변경했습니다.
- 무기 도감을 30종 구조로 재정리했습니다.
- 강화 성공률을 구간형으로 재설계했습니다.
  - +1~+10: 성공률 높음
  - +11~+20: 중간 난이도
  - +21~+30: 실패 체감 증가
  - +31 이상: 고위험 구간
- 강화 결과를 `성공 / 대성공 / 초월 성공 / 실패` 구조로 확장했습니다.
- 강화 실패 시 마력석 보상을 지급하도록 변경했습니다.
- 등급별 수익 배율을 크게 재정리했습니다.
- 판매 보상을 골드 + 보석 + 마력석 구조로 확장했습니다.
- 분해 기능은 제외하고 판매 중심 구조로 단순화했습니다.
- 가방 기능을 장착 / 판매 / 잠금 중심으로 정리했습니다.
- 뽑기 시 등급별 랜덤 옵션이 붙도록 개선했습니다.
- 옵션 도감을 추가했습니다.
- 성장 목표/성장 훈련 기능은 제거 상태를 유지합니다.

### 버전 규칙
- `0.x.y`: 정식 출시 전 개발 단계
- `0.69.0`: 큰 구조 변경 버전
- `0.69.2`: 이후 UI/버그 수정 버전
- `1.0.0`: 정식 공개 버전

## v63 주요 개선사항

### 1. 실제 게임 루프 개선

기존 흐름은 `클릭 → 강화 → 반복`에 가까웠습니다.  
v63은 아래 구조로 개선했습니다.

```text
클릭/자동 수익
→ 크리티컬/콤보 보상
→ 골드/보석/마력석 획득
→ 강화/옵션 파밍
→ 옵션/속성 강화
→ 더 빠른 성장
```

### 2. 핵심 HUD 재배치

모바일에서도 바로 확인할 수 있도록 핵심 수치를 메인 화면에 표시했습니다.

- 클릭 수익
- 초당 수익(DPS)
- 크리티컬 확률
- 콤보 배율
- 보석 확정 보상까지 남은 골드
- 강화 성공률

### 3. 크리티컬 시스템 추가

클릭 채굴 시 확률적으로 크리티컬이 발생합니다.

- 등급이 높을수록 기본 크리 확률 증가
- 화염/암흑 속성은 크리 쪽에 추가 이점
- 크리티컬 피해 옵션으로 배율 증가
- 크리티컬 발생 시 별도 메시지와 연출 표시

### 4. 콤보 시스템 추가

짧은 시간 안에 연속 터치하면 콤보 배율이 증가합니다.

- 빠른 클릭에 보상 제공
- 콤보 배율은 메인 HUD에 표시
- 클릭 채굴 손맛 강화

### 5. 성장 목표 시스템 제거

v68부터 성장 목표는 UI뿐 아니라 보상/진행도/저장 데이터 기능까지 제거했습니다.

저장 데이터에는 기존 목표 진행값이 남아 있어도 더 이상 사용하지 않습니다.

### 6. 랜덤 옵션 구조 확장

옵션 풀을 확장했습니다.

- 클릭 수익 증가
- 자동 수익 증가
- 강화 성공률 보너스
- 골드 획득량 증가
- 보석 획득량 증가
- 크리티컬 확률
- 크리티컬 피해

옵션은 가중치 기반으로 등장하며, 같은 등급의 마법봉이라도 성격이 달라지도록 구성했습니다.

### 7. 데이터 구조 변경

저장 키가 변경되었습니다.

```js
SAVE_KEY = "magicWandRpgSaveV0690"
SAVE_VERSION = 692
```

기존 저장 데이터는 자동으로 불러오며, 성장 목표 관련 값은 더 이상 사용하지 않습니다. 0.69.0부터 옵션 도감 데이터가 추가 저장되며, 0.69.2에서 모바일 UI/크리티컬 밸런스를 안정화했습니다.

추가 저장 항목:

- `totalClicks`
- `totalCrits`
- `bestCrit`
- `optionDex`

### 8. UI/UX 개선

- 메인 화면에서 성장 목표 카드 제거
- 현재 무기 정보에 초당 수익, 크리티컬, 전투력 표시
- 크리티컬 발생 시 시각 효과 추가
- 모바일에서 핵심 수치가 잘리지 않도록 반응형 보정

### 🟢 UI 개선

**5. compact-stats HUD 크리+콤보 셀 분리**

- 다크/라이트 테마 각각 적합한 배경/텍스트 색상 적용
- 퀘스트 완료 시 초록 테두리 + 바 색상 변경으로 완료 상태 시각화
