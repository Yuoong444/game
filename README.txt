v58 게임 기능/밸런스 개선 내용
- 초반 골드 획득량 상향: Lv.10 전까지 클릭 수익 보정 추가
- 등급별 골드 성장 체감 상향: D~SSS 등급 클릭 배율/기본 보너스 강화
- 강화비 부담 완화: 시작 강화비와 성공/실패 후 비용 증가폭 하향
- 강화 실패 부담 완화: Lv.10 전 실패 페널티 없음, 고레벨 페널티도 축소
- 강화 성공률 보정: 등급 특성 성공률 보너스가 실제 성공률에 반영되도록 개선
- 자동 수익 상향: 마력석/강화 단계가 자동 골드에 더 크게 반영
- 마력석 구매 효율 개선: 구매 시 마력석 +4, 다음 비용 증가폭 완화
- 성장/수련 보상 상향: 성장 단계 보상 골드 증가, 수련하기에 골드 보상 추가
- 보석 확정 보상 완화: 누적 골드 30,000당 1개 → 20,000당 1개
- 중복 마법봉 판매 골드 상향
