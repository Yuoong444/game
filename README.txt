마법봉 강화 RPG - 진짜 모바일 게임 UI 완성판

포함 파일:
- index.html
- style.css
- script.js
- README.txt

이번 버전 특징:
- 모바일 게임 앱처럼 보이는 HUD 상단 UI 적용
- 강화/도감/옵션/뽑기/가방 탭에 아이콘 적용
- 하단 액션 버튼을 모바일 스킬 버튼처럼 개선
- 마법봉 카드 배경, 오라, 마법진, 빛 효과 강화
- 가방 UI 카드형 개선
- 현재 마법봉 가방 저장/꺼내기 기능 유지
- 장착 중인 마법봉 판매 방지
- 모바일에서 하단 버튼이 잘리지 않도록 safe-area 대응
- PC에서도 가운데 정렬과 가독성 유지

실행 방법:
1. 압축을 풉니다.
2. index.html 파일을 브라우저로 실행합니다.
3. GitHub Pages에는 index.html, style.css, script.js, README.txt를 그대로 올리면 됩니다.

기존 플레이 기록이 브라우저 localStorage에 남아 있으면 이어서 플레이됩니다.
초기 상태로 테스트하려면 게임 내 초기화 버튼을 눌러주세요.
