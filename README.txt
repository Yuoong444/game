마법봉 강화 RPG v54 - 소스 정리판

적용 방법
1. ZIP 압축을 풀어 GitHub 저장소에 그대로 업로드합니다.
2. 기존 파일은 덮어쓰기 하면 됩니다.
3. GitHub Pages 반영 후 브라우저 캐시가 남으면 새로고침(Ctrl+F5)을 해주세요.

v54 정리 내용
- 밝은 테마 / 다크 테마 전환 기능 유지 및 저장 키 v54로 정리
- v53 이전의 메뉴 버튼 중복 이벤트 구조 정리
- 모바일 메뉴 열기 / 탭 패널 열기 / 닫기 흐름을 bindUiEvents()로 통합
- 메뉴에서 탭을 열면 메뉴가 닫히고, 패널 닫기 버튼으로 메인 화면 복귀
- PC에서는 우측 패널이 일반 탭 방식으로 표시되도록 유지
- 저장 데이터는 v53 이하 키에서 자동 마이그레이션
- index.html 캐시 버전 v54로 변경
- JS 문법 체크 통과
- HTML 중복 ID 점검 완료

주의
- style.css는 기존 상용 UI 효과가 많이 포함되어 있어 전체 삭제형 리팩토링은 하지 않았습니다.
- 대신 v54 최종 override를 추가해 메뉴/패널/테마 충돌 가능성을 줄였습니다.
