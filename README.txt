마법봉 강화 RPG - v8 무기 이미지 + 뽑기 연출 버전

변경 내용:
- 마법봉을 CSS 도형이 아닌 실제 SVG 이미지로 교체
- 등급별 무기 이미지 추가: E, D, C, B, A, S, SS, SSS
- 현재 장착 무기의 등급에 따라 메인 무기 이미지 자동 변경
- 뽑기 화면에 포탈/빛/무기 등장 연출 추가
- S/SS/SSS 등급 뽑기 시 전용 배너 연출 추가
- 도감 카드와 가방 목록에도 무기 썸네일 표시
- 기존 기능 유지: 도움말, 보석 획득, 등급별 클릭 골드, 가방, 판매, 강화비용 표시

업로드 방법:
GitHub Pages에 아래 항목을 그대로 올려주세요.
- index.html
- style.css
- script.js
- README.txt
- img 폴더 전체

주의:
img 폴더를 빠뜨리면 무기 이미지가 표시되지 않습니다.
