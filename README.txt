마법봉 강화 RPG - 상용 게임 UI v9

변경 내용:
- 등급 글자가 들어간 샘플 무기 이미지를 제거하고 순수 무기 이미지로 교체
- PC/모바일 우측 메뉴 깨짐 방지
- 하단 강화/마력석 버튼 고정 UI 개선
- 가방 목록이 하단 버튼에 가리지 않도록 여백 조정
- 뽑기 포탈/빛/등장 연출 강화
- 전체 패널, 버튼, HUD를 상용 모바일 RPG 느낌으로 재정리
